Tiny Pets — Animated Pixel Cartoon Pet Pack
================================================

12 cute animated pets, each with 4 animations (idle, jump, sleep, happy),
in two sizes (32px and 64px).

FILES
  <Pet>/<pet>_<anim>_<size>.png   Horizontal spritesheet, 8 frames.
  <Pet>/<pet>.json                Frame size, fps and file map.

USAGE
  Each spritesheet is 8 frames laid out horizontally.
  Frame size = the sheet height (32 or 64). Slice by frame width = height.
  Suggested FPS: idle 8 · jump 12 · sleep 6 · happy 10.
